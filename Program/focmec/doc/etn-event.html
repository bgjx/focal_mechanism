FOCMEC applied to a data set
of P-wave polarities

This page supplements the discussion in Section 4.1 of the FOCMEC manual: the application of the FOCMEC package to find the focal mechanism for a magnitude 4.2 event using P-wave polarities at 37 local-to-regional-distance stations. Directory ./focmec/sample_runs/ETNEvent includes a README file, drivers, and input files for the analysis. Copies of output files are in directory ./focmec/doc/ETNEvent-doc/

The driver files for programs focmec and focplt are Unix csh scripts that include comments. The programs can be run interactively without using scripts, but using the scripts allows quick editing to fine-tune runs. The focplt driver scripts end with a conversion from the SGF formal used in the package for plots to a PDF file (with a tight bounding box).

See ./focmec/sample_runs/ETNEvent/README.txt for instructions and information about the driver scripts.
